# Chat GPT Local 100% Privado, Gratis

Este proyecto integra un modelo de inteligencia artificial en una aplicación web que se ejecuta completamente en el navegador. Utiliza la biblioteca MLCEngine para interactuar con un modelo de lenguaje avanzado y proporcionar una experiencia de chat en tiempo real.

## Características

- **Modelo de Lenguaje Avanzado**: Utiliza el modelo `Llama-3-8B-Instruct-q4f32_1-MLC-1k` para generar respuestas inteligentes y contextuales.
- **Interfaz de Usuario**: Ofrece una interfaz de chat simple con mensajes tanto del usuario como del asistente.
- **Ejecución en el Navegador**: Todo el procesamiento se realiza en el navegador utilizando Web Workers, sin necesidad de servidores backend.
- **Privacidad Completa**: La aplicación se ejecuta localmente en tu navegador, asegurando que tus datos permanezcan privados.
